# Bank_Loan_Project
📊 Bank Loan Analysis Dashboard (Power BI)

This project presents an interactive Bank Loan Analysis Dashboard built using Power BI. It provides insights into loan applications, approvals, and customer behavior to support data-driven decision-making.

📌 Project Overview
Developed an interactive dashboard to analyze bank loan data
Visualized key metrics like loan approval rate, total applications, and loan distribution
Enabled data-driven insights for financial decision-making
📊 Key Features
📈 Loan Application Trends (monthly/yearly analysis)
💰 Total Loan Amount & Funded Amount tracking
✅ Loan Approval vs Rejection Analysis
👥 Customer Segmentation (based on income, employment, etc.)
🌍 Regional/Location-based loan distribution
📉 Default risk & loan status insights
📂 Dashboard Insights

The dashboard helps answer:

How many loan applications are received?
What is the approval rate?
Which customer segments receive more loans?
Which regions have higher loan demand?
What factors impact loan approval?
🛠️ Tools & Technologies
